# References and Resources

## Comprehensive Bibliography

This section provides comprehensive references, additional learning resources, and external links to support your learning journey in humanoid robotics.

## üìö Core Robotics References

### Foundational Textbooks

1. **"Probabilistic Robotics"** - Thrun, Burgard, and Fox (2005)
   - Essential reading for probabilistic methods in robotics
   - Covers localization, mapping, and SLAM algorithms

2. **"Modern Robotics: Mechanics, Planning, and Control"** - Lynch and Park (2017)
   - Comprehensive treatment of robot kinematics and dynamics
   - Includes practical examples and code implementations

3. **"Robotics, Vision and Control: Fundamental Algorithms in MATLAB"** - Peter Corke (2017)
   - Practical guide with MATLAB implementations
   - Excellent for computer vision in robotics

4. **"Planning Algorithms"** - Steven M. LaValle (2006)
   - Comprehensive coverage of motion planning techniques
   - Essential for autonomous robot navigation
